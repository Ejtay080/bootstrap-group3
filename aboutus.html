About Us


The EDE Engine is a game designer engine specifically used for 2D games. The tools it provides were made to be highly user friendly and can be personalized to the user's preference. Other reasons to download and use EDE can be found below

Easy to Use for Everyone


EDE's has its own built-in code-editor that is highly user-friendly and easy to use. With it, the user can create files, edit code, and store all their information and files in one place.

Create Sprites and Animations


EDE also has tools for making sprites, spritesheets, and animations. If the user has their own files regarding sprites and animations, they can be uploaded and stored into EDE.

Documentation Made Easy


Any changes to a project will be documented by EDE. This is to help teams or solo developers keep track of changes made and will show who made the change and when the change was made.
