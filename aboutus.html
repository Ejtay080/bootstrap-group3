About us


The GDE Engine is a game designer engine specifically for 2D games. The tools it provides were made to be highly user friendly and can be personalized to the user's preference. Other reasons to download and use EDE can be found below

Easy to Use for Everyone


GDE's has its own built-in code-editor that is highly user-friendly and easy to use. With it, the user can create files, edit code, and store all their information and files in one place.

Create sprites and Animations


GDE also has tools for making sprites, spritesheets, and animations. If the user has their own files regarding sprites and animations, they can be uploaded and stored into GDE.
